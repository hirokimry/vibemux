# vibemux MVV

---

## Mission

ターミナルを、バイブコーディングの最高の場にする

## Vision

AIと人が同じ画面で息を合わせて開発する世界

## Values

1. **一発で整う** — コマンドひとつで開発環境が完成する。セットアップに時間を使わせない
2. **フローを切らない** — ウィンドウ切り替えゼロ。ファイル・git・AIが常に視界にある
3. **道具に縛られない** — Claude Code、Aider、何でも載る。特定ツールに依存しない
4. **組織ごとAIで回す** — 開発も品質管理も、AIエージェントが自律的に動く
